Cuộc thi Web Design

Web Design là cuộc thi được tổ chức vào quý 1 hằng năm. Cuộc thi xoay quanh chủ đề thiết kế website bằng ngôn ngữ HTML, CSS và Javascript. Là sân chơi mới giúp các bạn làm quen, học hỏi và cải thiện kỹ năng lập trình web.
